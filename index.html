Roadmap Belajar Fullstack

Ikuti tahapan belajar ini untuk menjadi seorang Fullstack Developer.

  • Step 1

    Web Development

    Mulailah dengan mempelajari dasar-dasar pengembangan web: HTML, CSS, dan JavaScript. Tahap ini membangun fondasi yang kuat untuk memahami teknologi web.

  • Step 2

    Frontend Development

    Setelah dasar, pelajari React.js, library populer untuk membuat UI yang dinamis dan modern. Anda akan memahami konsep seperti component-based architecture, state management, dan routing.

  • Step 3

    Backend Development

    Pelajari backend development menggunakan Node.js dan framework Express.js. Kuasai pengelolaan database dengan PostgreSQL dan NoSQL. Di sini, Anda akan memahami bagaimana membangun API dan mengelola server.

  • Step 4

    Mobile Development

    Perluas keahlian Anda dengan membangun aplikasi mobile menggunakan React Native. Anda akan belajar membuat aplikasi lintas platform (iOS & Android) dengan efisien menggunakan JavaScript.

  • Jadi
    Fullstack
    Developer!

HTML & CSS

Belajar dasar-dasar pembuatan halaman web.

Materi ini mencakup pembuatan struktur HTML, penggunaan elemen-elemen dasar seperti tag, form, table, dan lainnya, serta styling dengan CSS untuk memperindah tampilan web.

  • Topik: HTML, CSS, Layouting, Styling
  • Durasi: 2 Minggu

JavaScript

Memahami dasar dan konsep JavaScript.

Materi ini membahas JavaScript untuk interaktivitas web, termasuk variabel, kondisi, perulangan, event handling, dan DOM manipulation untuk membuat halaman web dinamis.

  • Topik: JavaScript, Event Handling, DOM Manipulation
  • Durasi: 3 Minggu

React Js

Membangun aplikasi web dengan React JS.

Materi ini mengajarkan pembuatan aplikasi web modern menggunakan React, termasuk penggunaan component, state, props, dan hooks.

  • Topik: React, Components, State, Props, Hooks
  • Durasi: 3 Minggu

Node JS

Membangun backend dengan Node.js.

Belajar cara membuat server backend menggunakan Node.js dengan Express, serta cara menghubungkan aplikasi dengan database dan menangani request HTTP.

  • Topik: Node.js, Express, HTTP Request, REST API
  • Durasi: 4 Minggu

SQL & NoSQL

Memahami basis data relasional dan non-relasional.

Materi ini mencakup SQL untuk pengelolaan basis data relasional dan NoSQL seperti MongoDB untuk data berbasis dokumen, serta cara melakukan query dan mengelola data.

  • Topik: SQL, NoSQL, MongoDB, Query
  • Durasi: 3 Minggu

Materi Lainnya

Pelajari topik tambahan yang berguna.

Materi lainnya meliputi pengembangan keterampilan tambahan seperti version control (Git), deployment aplikasi, pengujian aplikasi, dan lainnya.

  • Topik: Git, Deployment, Testing, DevOps
  • Durasi: 2 Minggu
